# 如何开始 SuperTextView ？ 
在你的**build.gradle**中加入：

```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}

dependencies {
    compile 'com.github.chenBingX:SuperTextView:VERSION_CODE'
}
```

[👉 点击这里，查看真实发行版本号, 替换 VERSION_CODE](https://github.com/chenBingX/SuperTextView/releases)
